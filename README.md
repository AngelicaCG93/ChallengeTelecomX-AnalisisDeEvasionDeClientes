# ChallengeTelecomX-AnalisisDeEvasionDeClientes

📊 Análisis de Evasión de Clientes
📌 Descripción del Proyecto
Este proyecto tiene como objetivo analizar los factores que influyen en la evasión de clientes (Churn) en una empresa de telecomunicaciones.
La evasión representa a los clientes que cancelan el servicio, lo que impacta directamente en los ingresos de la empresa. A través de análisis exploratorio de datos, se identificaron patrones y variables asociadas a la cancelación.
🛠 Herramientas Utilizadas
Python
Pandas
Matplotlib
Seaborn
Google Colab
📊 Análisis Realizado
✔ Limpieza y tratamiento de datos
✔ Transformación de variables
✔ Análisis descriptivo
✔ Distribución de la variable Churn
✔ Análisis de variables categóricas (contrato, género, método de pago)
✔ Análisis de variables numéricas (antigüedad y cargos mensuales)
🔎 Principales Hallazgos
Los contratos mes a mes presentan mayor tasa de evasión.
Los clientes con menor antigüedad tienden a cancelar más.
Los cargos mensuales elevados están asociados con mayor evasión.
El género no muestra diferencias significativas en la cancelación.
💡 Recomendaciones
Incentivar contratos de largo plazo.
Implementar estrategias de fidelización para clientes nuevos.
Evaluar la estructura de precios.
Aplicar campañas de retención dirigidas a clientes con mayor riesgo de evasión.
